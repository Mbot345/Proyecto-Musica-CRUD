# Proyecto-Musica-CRUD
Sistema CRUD desarrollado en Python y SQL Server para la gestión de canciones de la base de datos del proyecto ViveEC, utilizando programación orientada a objetos, procedimientos almacenados y configuración mediante archivos JSON.

## Tecnologías Utilizadas
- Python
- SQL Server
- pyodbc
- JSON

## Funcionalidades
- Insertar canciones
- Consultar canciones
- Actualizar canciones
- Eliminar canciones

## Características del Proyecto
- Programación Orientada a Objetos
- Uso de procedimientos almacenados
- Manejo de vistas SQL
- Configuración de conexión mediante archivo JSON
- Validación de datos ingresados por el usuario

## Archivos Incluidos
- `Ejercicio_2_Python-SQLServer.py` → Código principal del sistema CRUD orientado a objetos
- `config.json` → Configuración de conexión a SQL Server
- `Script_Configuracion_DB_Cancion.sql` → Script completo de creación de base de datos, objetos programables, login, usuario y permisos
- `README.md` → Documentación del proyecto
